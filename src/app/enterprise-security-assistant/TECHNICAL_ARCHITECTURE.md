## 1. 技术选型

### 1.1 核心技术栈

| 技术领域 | 选型方案 | 理由 |
|---------|---------|------|
| Agent框架 | LangChain + LangGraph | 成熟生态、工具丰富、社区活跃 |
| 工作流引擎 | LangGraph | 状态机编排、可视化调试 |
| LLM | OpenAI GPT-4 / DeepSeek | 性能领先、上下文长度长 |
| 向量数据库 | Chroma / FAISS | 轻量级、易部署、Python原生 |
| 消息队列 | Redis Stream | 高性能、持久化、支持消费者组 |
| 状态存储 | Redis | 高性能、支持复杂数据结构 |
| 主数据库 | PostgreSQL 14+ | 支持JSONB、窗口函数、成熟稳定 |
| 缓存 | Redis | 高性能、支持分布式锁 |
| API框架 | FastAPI | 高性能、类型安全、自动文档 |
| 任务队列 | Celery + Redis | 成熟稳定、监控完善 |
| 监控 | Prometheus + Grafana | 云原生标准、生态完善 |
| 日志 | ELK Stack | 成熟方案、查询能力强 |
| 链路追踪 | Jaeger / Zipkin | OpenTelemetry标准、分布式追踪 |
| 配置中心 | Apollo / 环境变量 | 支持多环境、动态配置 |

### 1.2 LLM模型选型策略

| 任务类型 | 推荐模型 | 特点 | 成本（每1K tokens） |
|---------|---------|------|-------------------|
| 意图识别 | GPT-4o-mini | 快速、准确 | $0.00015 |
| 日志查询解释 | GPT-4o | 逻辑推理能力强 | $0.005 |
| 评分解读 | GPT-4o | 知识检索+生成 | $0.005 |
| 威胁分析 | GPT-4o-32K | 长上下文、复杂推理 | $0.01 |
| 合规检查 | GPT-4o | 规则验证 | $0.005 |
| 知识库检索 | DeepSeek-Chat | 长上下文、低成本 | $0.001 |
| 异常检测 | GPT-4o | 代码生成 | $0.005 |

**模型路由决策树**：
```
用户请求
    │
    ├─ 意图识别？ → GPT-4o-mini（快速）
    │
    ├─ 简单查询？ → GPT-4o（中等成本）
    │
    ├─ 复杂分析？
    │   ├─ 长上下文？ → GPT-4o-32K
    │   └─ 常规分析？ → GPT-4o
    │
    └─ 成本敏感？ → DeepSeek-Chat
```

```

    │
```

```
    │   └─ 目标：ThreatAgent

    │   ├─ 关键词：合规、检查、审计、GDPR、等保
```
